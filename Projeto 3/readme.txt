Navegue até o diretório do projeto

Se existir o arquivo imprima.txt apague

Compile utilizando o makefile e execute no terminal:

make

./exec

A saída apresentando os emparelhamentos será dada no arquivo imprima.txt

Considere, na saída: Nos emparelhamentos: Nas escolas (começando de E1 até E50), se não aparece "prof2", não há um professor naquela posição, ou aquela posição não existe para a escola. Caso apareça "prof1:" e "prof2:" vazios, significa que aquela posição não possui um professor, o mesmo foi substituiído por uma ligação melhor durante o tempo de execução.(editado)
